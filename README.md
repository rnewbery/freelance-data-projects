# Freelance Data & Dashboard Projects

This repository contains examples of practical data analysis, reporting and dashboard work created for portfolio purposes.

The focus is on operational reporting, livestock performance monitoring, spreadsheet-to-dashboard workflows, and clear visual summaries for non-technical users.

## Projects

### Livestock Power BI Dashboards

A set of anonymised Power BI dashboard examples focused on calf and livestock performance monitoring.

The dashboards demonstrate:

* Weight progression tracking
* Milk consumption analysis
* Treatment group comparison
* Pen/location performance comparison
* Dairy or farm origin breakdowns
* Batch-level reporting
* Practical KPI summaries for operational decision-making

View project: `livestock-powerbi-dashboards/`

## Tools Used

* Power BI
* Excel
* SQL
* Data cleaning and transformation
* Dashboard design
* Operational reporting

## Data Privacy

All examples are anonymised and use either sample data, recreated data, or screenshots with sensitive details removed. No confidential farm, client, animal, supplier or commercial data is included.
